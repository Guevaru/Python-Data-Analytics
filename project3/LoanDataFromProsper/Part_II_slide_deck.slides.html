Investigation Overview¶

In this project,I will investigate the 113,937 rows and 81 columns long Loan dataset from Prosper. I'll choose a few variables that are associated with Default to examine.

After some data preparation and cleaning, I will use EDA and various charts to examine the links between and among the variables in an effort to provide meaningful research findings.

Dataset Overview¶

There are 81 columns and 113,937 rows in the Prosper Loan dataset. The majority of variables are numerical, but others are category, boolean, and date. It is possible to rank factor variables with the following levels using variables like ProsperRating (Alpha) and IncomeRange.

I'll need to deal with some of the variables that contain a lot of null values, such prosperrating and prosperscore.

Default Loans Percentage¶

Using the loanstatus variable as a base, I generated a new variable called Default. Thus, 1 denotes a loan that has defaulted and 0 denotes a loan that has been repaid.
